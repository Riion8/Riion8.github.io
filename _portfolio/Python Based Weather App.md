# Requirements

To use this repository please ensure that you have Python install with
the following modules.

-   Requests

-   Json

-   Columnar

-   Pandas

-   OS

-   Datetime

You will also need an API key from Open Weather Map. They have a free
option to get an API key. Please use the link below to sign up for a
free key if you do not have one.

<https://home.openweathermap.org/users/sign_up>

[Repository](https://github.com/Riion8/Data-Science-Portfolio/tree/main/Weather%20App)

# Contents

WeatherAppNoAPIKey.py -- Weather App

USStateAbbreviations.csv -- State Names to Abbreviations mapping file

# Usage

1.  Input your API key into the variable "apikey"

2.  Run the python code

3.  Follow on screen prompts to get the weather

# Summary

At a high level this app takes in either a city/state or zip code and
returns the current weather and 5-day forecast.
